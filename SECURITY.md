# Security

Guvenlik bildirimi icin GitHub issue acmak yerine proje sahibine dogrudan ulasin.

## Desteklenen Surum

Bu starter aktif olarak `main` dali uzerinden gelistirilir.

## Varsayilanlar

- API `x-powered-by` header'ini kapatir.
- JSON body limiti `1mb` olarak ayarlidir.
- `npm audit --omit dev` CI kalite kapisinin bir parcasi olarak calistirilabilir.
