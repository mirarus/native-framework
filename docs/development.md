# Gelistirme Rehberi

## Gunluk Komutlar

```bash
npm install
npm run check
npm run dev
```

## Kalite Kapisi

```bash
npm run lint
npm run typecheck
npm run test
npm run build
npm run ci
```

## Hedefler

```bash
npm run dev:api
npm run dev:web:react
npm run dev:web:vanilla
npm run dev:mobile
npm run dev:desktop
```

## Notlar

- Node.js 22.12+ gerekir.
- Build ciktilari `apps/*/dist` altina gider ve git'e alinmaz.
- `npm run clean` build ciktilarini temizler.
- `npm run info` framework manifest ve port bilgisini yazdirir.
