# Android Target

Bu klasor Android hedefinin nasil olusturulacagini anlatir.

Native Android projesini uretmek icin:

```bash
npm --workspace fullstack-uniframe run android:add
npm --workspace fullstack-uniframe run android:sync
npm --workspace fullstack-uniframe run android:open
```

Capacitor native Android klasorunu example proje kokunde `android/` olarak uretir. Android Studio ve Android SDK gerekir.
